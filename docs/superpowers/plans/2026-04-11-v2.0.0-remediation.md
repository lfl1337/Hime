**Goal:** Resolve all 5 Critical Issues and 8 of 10 Warnings from `reports/verification_20260411_0232/FINAL_REPORT.md`, build Vitest + Pytest test infrastructure, and deliver a fully working Chrome-MCP-verified Pipeline v2 dry-run end-to-end — reaching v2.0.0 release readiness.

**Architecture:** Phased remediation with strict HALT-gates after each phase. Every phase writes a standalone sub-report; Luca confirms `Proceed with Phase N` before the next starts. ## 🚨 GUARDRAILS — Non-Negotiable (read before starting)

**Read first:** Before any action, read `reports/verification_20260411_0232/FINAL_REPORT.md` in full and reconcile against the Issue Matrix below. Any deviation (new bugs, already-fixed items) is documented at the start of Phase 0 output.

**HALT-Gates:** Every phase ends with a mandatory `HALT — wait for 'Proceed with Phase N+1'`. No phase crosses into the next without explicit user confirmation. This is enforced by the phase's final step.

**No git mutations:** Zero commits, zero pushes, zero branch switches during remediation. `git status` / `git diff` / `git log` are allowed. Final commits happen only after Phase 9 upon Luca's explicit request. **This plan contains NO `git commit` steps** — phase reports replace commits as the persistence mechanism.

**No training runs:** Not even a 10-step sanity training. Training scripts are extended and config-validated; no `trainer.train()` calls are made in this plan.

**No downloads without approval:** Phase 2 asks explicitly before each download (~17 GB total). Phase 2 cannot proceed past its approval gate without a `Proceed with download` response.

**Test ports:** All new test services bind to `127.0.0.1` on ports ≥ 23420:
- Test Backend: `23420`
- Test Vite Dev: `23421`
- **Never** use production ports `1420` or `18420–18430`.

**Test DB isolation:** All tests use `sqlite:///:memory:` or a `tmp_path` fixture DB. Production DBs (`hime.db`, `app/backend/hime.db`) are read-only to this plan.

**Error budget:** If a phase hits >3 unexpected errors, STOP, document state in the phase report, ask Luca.

**Chrome MCP:** Used continuously after Phase 0 baseline — every phase that touches UI or backend routes runs the Standard Walkthrough Sequence (defined below) and logs screenshots + console errors. New console errors vs. baseline = phase failure.

**Report destination:** All per-phase reports land in `reports/remediation_YYYYMMDD_HHMM/<phase>.md` where the timestamp is computed in Phase 0. The final `REMEDIATION_REPORT.md` is the roll-up.

"""
Pipeline v2 dry-run stubs.

Every real model class gets a DryRunModel counterpart that implements the
same interface (load / unload / generate / review / aggregate) but performs
no VRAM allocation and returns deterministic fake output. Used by tests
and UI verification runs triggered with HIME_DRY_RUN=1.
"""
from __future__ import annotations

import asyncio
import hashlib
from typing import Any

from pydantic import BaseModel

from .stage1._types import Stage1Drafts
from .stage4_aggregator import AggregatorVerdict
from .stage4_reader import PERSONAS, PersonaAnnotation


class DryRunModel:
    """Generic dry-run stub matching the load/unload/generate surface of real models."""

    def __init__(self, name: str) -> None:
        self.name = name
        self.loaded = False

    def load(self, settings: Any) -> None:
        self.loaded = True

    def unload(self) -> None:
        self.loaded = False

    def generate(self, prompt: str) -> str:
        # Deterministic: hash-based suffix so repeated calls produce identical output
        digest = hashlib.sha1(prompt.encode("utf-8", "ignore")).hexdigest()[:8]
        snippet = prompt[:40].replace("\n", " ")
        return f"[DRY-RUN {self.name}:{digest}] {snippet}"


# --- Stage 1 -----------------------------------------------------------------

async def make_dry_run_stage1_drafts(
    *,
    segment: str,
    rag_context: str,
    glossary_context: str,
) -> Stage1Drafts:
    """Produce deterministic Stage1Drafts without loading any Stage 1 model."""
    _ = rag_context, glossary_context  # unused in dry-run
    digest = hashlib.sha1(segment.encode("utf-8", "ignore")).hexdigest()[:8]
    snippet = segment[:30].replace("\n", " ")

    def stamp(tag: str) -> str:
        return f"[DRY-RUN stage1/{tag}:{digest}] {snippet}"

    return Stage1Drafts(
        qwen32b=stamp("qwen32b"),
        translategemma12b=stamp("translategemma"),
        qwen35_9b=stamp("qwen35_9b"),
        gemma4_e4b=stamp("gemma4_e4b"),
        jmdict={"terms": []},
    )


# --- Stage 2 / Stage 3 -------------------------------------------------------

async def dry_run_stage2_merge(
    drafts: dict[str, Any],
    rag_context: str,
    glossary_context: str,
) -> str:
    """Deterministic stage 2 merge without loading TranslateGemma-27B."""
    _ = rag_context, glossary_context
    base = drafts.get("qwen32b") or next((v for v in drafts.values() if isinstance(v, str) and v), "")
    digest = hashlib.sha1(base.encode("utf-8", "ignore")).hexdigest()[:8]
    return f"[DRY-RUN stage2:{digest}] {base[:40]}"


async def dry_run_stage3_polish(
    merged: str,
    glossary_context: str,
    retry_instruction: str = "",
) -> str:
    """Deterministic stage 3 polish without loading Qwen3-30B-A3B."""
    _ = glossary_context
    retry_suffix = f" retry={retry_instruction[:20]}" if retry_instruction else ""
    digest = hashlib.sha1((merged + retry_instruction).encode("utf-8", "ignore")).hexdigest()[:8]
    return f"[DRY-RUN stage3:{digest}]{retry_suffix} {merged[:40]}"


# --- Stage 4 -----------------------------------------------------------------

class DryRunStage4Reader:
    """Dry-run counterpart of Stage4Reader.

    Produces 15 persona annotations per sentence without loading Qwen3.5-2B.
    """

    def __init__(self) -> None:
        self.loaded = False

    def load(self, settings: Any) -> None:
        self.loaded = True

    def unload(self) -> None:
        self.loaded = False

    async def review(
        self,
        *,
        sentences: list[str],
        source_sentences: list[str],
    ) -> list[PersonaAnnotation]:
        # yield to the event loop so callers awaiting us don't see blocking behavior
        await asyncio.sleep(0)
        out: list[PersonaAnnotation] = []
        for sid, (translation, source) in enumerate(zip(sentences, source_sentences)):
            digest = hashlib.sha1(translation.encode("utf-8", "ignore")).hexdigest()[:8]
            for persona_name, _focus in PERSONAS:
                # deterministic rating = 0.80 so every sentence passes (no retry loop spin)
                out.append(PersonaAnnotation(
                    persona=persona_name,
                    sentence_id=sid,
                    rating=0.80,
                    issues=[],
                    suggestion=f"[DRY-RUN {persona_name}:{digest}]",
                ))
        return out


class DryRunStage4Aggregator:
    """Dry-run counterpart of Stage4Aggregator — always returns 'okay'."""

    def __init__(self) -> None:
        self.loaded = False

    def load(self, settings: Any) -> None:
        self.loaded = True

    def unload(self) -> None:
        self.loaded = False

    async def aggregate(self, annotations: list[PersonaAnnotation]) -> AggregatorVerdict:
        await asyncio.sleep(0)
        sentence_id = annotations[0].sentence_id if annotations else 0
        return AggregatorVerdict(
            sentence_id=sentence_id,
            verdict="okay",
            retry_instruction=None,
            confidence=0.80,
        )


def make_dry_run_stage4_reader() -> DryRunStage4Reader:
    return DryRunStage4Reader()


def make_dry_run_stage4_aggregator() -> DryRunStage4Aggregator:
    return DryRunStage4Aggregator()

## Phase 5 — Training v2 + Curriculum Activation (C3, C5)

**Goal:** Refactor `scripts/train_generic.py` from a monolithic `MODEL_CONFIGS` dict + dispatcher into a modular plugin architecture, add Pipeline-v2 models (TranslateGemma-12B, Qwen3.5-9B, Qwen3-30B-A3B), add a `--validate-config` flag that exercises config loading without touching VRAM, and activate the curriculum-learning block in `scripts/training_config.json`.

**Constraint:** The spec explicitly forbids any actual training runs in this session — not even a 10-step smoke test. All verification is static: `--help`, `--validate-config`, tokenizer compat.

### 🟢 Architecture decision (Luca 2026-04-11, Phase 0 answer)

**Modular rewrite is MANDATORY**, not optional. The verification report explicitly identified `train_generic.py` as "monolithisch mit hardcodierten MODEL_CONFIGS und nicht modular erweiterbar" — extending it in-place makes the problem worse. Plugin structure:

```
scripts/training/
├── configs/
│   ├── __init__.py              # registry lookup: get_config(key) -> Config
│   ├── qwen32b.py               # v1, kept for backward compat
│   ├── qwen14b.py               # v1
│   ├── qwen72b.py               # v1
│   ├── gemma27b.py              # v1
│   ├── deepseek.py              # v1
│   ├── translategemma12b.py     # v2 (NEW)
│   ├── qwen35_9b.py             # v2 (NEW)
│   └── qwen3_30b_a3b.py         # v2 (NEW, MoE)
└── trainers/
    ├── __init__.py              # registry lookup: get_trainer(backend) -> Trainer
    ├── unsloth_trainer.py       # for qwen*, gemma27b, deepseek, qwen35_9b, qwen3_30b_a3b
    └── transformers_trainer.py  # for translategemma12b (chat template must not be overwritten)
```

`scripts/train_generic.py` becomes a **thin dispatcher** that parses CLI args, looks up the config and trainer, and delegates. The legacy `MODEL_CONFIGS` dict at module level MAY remain as a deprecated shim that delegates into `scripts.training.configs.get_config(...)` — this keeps any external caller working — but the source of truth moves into the plugin files.

**Hard requirement: backward compatibility for `checkpoint-12400`.** The `qwen32b` adapter training path (LoRA rank 16, max_seq 1024, grad_accum 8, etc.) must still work end-to-end after the rewrite. This is enforced by a mandatory parametrized test (see Task 5.2b below) that instantiates each v1 config and verifies every field the original dict defined.

**What to do with the existing `MODEL_CONFIGS` dict:** delete it from `train_generic.py` after moving its contents into the per-model files. If a deprecated shim is kept for external callers, it must be a dynamic lookup (`MODEL_CONFIGS = _LazyConfigsShim()`) — not a duplicated static dict (that would reintroduce the drift problem).

"""Trainer plugin registry.

scripts/training/trainers/ that exposes a run(config, args) function.

from __future__ import annotations

from typing import Callable, Protocol

class Trainer(Protocol):

    if name not in _REGISTRY:
        raise KeyError(f"Unknown trainer backend: {name}. Known: {sorted(_REGISTRY)}")

```python
"""
hime_data.py — training data registry CLI.

Commands:
  register <path> --id <id> --kind <kind> --source <name> [--quality-field <field>]
  list
  export [--min-score <float>] --out <path>

The registry is `data/registry.jsonl` at the repo root. Each line is a JSON
object describing one training data file. See docs/superpowers/plans/
2026-04-11-v2.0.0-remediation.md Phase 6 for the schema.
"""
from __future__ import annotations

import argparse
import json
import os
import sys
from datetime import datetime, timezone
from pathlib import Path


def _registry_path(repo_root: Path) -> Path:
    return repo_root / "data" / "registry.jsonl"


def _load_registry(path: Path) -> list[dict]:
    if not path.exists():
        return []
    out: list[dict] = []
    for line in path.read_text(encoding="utf-8").splitlines():
        line = line.strip()
        if line:
            out.append(json.loads(line))
    return out


def _save_registry(path: Path, entries: list[dict]) -> None:
    path.parent.mkdir(parents=True, exist_ok=True)
    path.write_text(
        "\n".join(json.dumps(e, ensure_ascii=False) for e in entries) + "\n",
        encoding="utf-8",
    )


def _measure_jsonl(file_path: Path, quality_field: str | None) -> tuple[int, tuple[float, float] | None]:
    """Return (line_count, (min_score, max_score)) for a JSONL file."""
    count = 0
    min_s: float | None = None
    max_s: float | None = None
    with file_path.open("r", encoding="utf-8") as f:
        for line in f:
            line = line.strip()
            if not line:
                continue
            count += 1
            if quality_field:
                try:
                    obj = json.loads(line)
                    val = obj.get(quality_field)
                    if isinstance(val, (int, float)):
                        if min_s is None or val < min_s:
                            min_s = float(val)
                        if max_s is None or val > max_s:
                            max_s = float(val)
                except Exception:  # noqa: BLE001
                    continue
    score_range = None
    if min_s is not None and max_s is not None:
        score_range = (min_s, max_s)
    return count, score_range


def cmd_register(args: argparse.Namespace, repo_root: Path) -> int:
    file_path = (repo_root / args.path).resolve()
    if not file_path.exists():
        print(f"error: file not found: {file_path}", file=sys.stderr)
        return 2
    if not str(file_path).startswith(str(repo_root)):
        print(f"error: file outside repo root: {file_path}", file=sys.stderr)
        return 2

    registry = _registry_path(repo_root)
    entries = _load_registry(registry)
    if any(e["id"] == args.id for e in entries):
        print(f"error: id already exists in registry: {args.id}", file=sys.stderr)
        return 3

    count, score_range = _measure_jsonl(file_path, args.quality_field or None)
    entry = {
        "id": args.id,
        "path": str(file_path.relative_to(repo_root)).replace("\\", "/"),
        "kind": args.kind,
        "source": args.source,
        "lines": count,
        "quality_field": args.quality_field or "",
        "quality_range": list(score_range) if score_range else [],
        "added": datetime.now(timezone.utc).isoformat(),
        "notes": args.notes or "",
    }
    entries.append(entry)
    _save_registry(registry, entries)
    print(f"[OK] registered {args.id}: {count} lines, score={score_range}")
    return 0


def cmd_list(args: argparse.Namespace, repo_root: Path) -> int:
    registry = _registry_path(repo_root)
    entries = _load_registry(registry)
    if not entries:
        print("(empty)")
        return 0
    print(f"{'id':<32} {'kind':<22} {'lines':>10}  path")
    print("-" * 80)
    for e in entries:
        print(f"{e['id']:<32} {e['kind']:<22} {e['lines']:>10}  {e['path']}")
    return 0


def cmd_export(args: argparse.Namespace, repo_root: Path) -> int:
    registry = _registry_path(repo_root)
    entries = _load_registry(registry)
    out_path = Path(args.out).resolve()
    out_path.parent.mkdir(parents=True, exist_ok=True)

    written = 0
    with out_path.open("w", encoding="utf-8") as out_f:
        for entry in entries:
            src = (repo_root / entry["path"]).resolve()
            if not src.exists():
                print(f"[warn] source missing, skipped: {src}", file=sys.stderr)
                continue
            qf = entry.get("quality_field") or ""
            with src.open("r", encoding="utf-8") as in_f:
                for line in in_f:
                    line = line.rstrip("\n")
                    if not line.strip():
                        continue
                    if args.min_score is not None and qf:
                        try:
                            obj = json.loads(line)
                            val = obj.get(qf)
                            if not isinstance(val, (int, float)) or val < args.min_score:
                                continue
                        except Exception:  # noqa: BLE001
                            continue
                    out_f.write(line + "\n")
                    written += 1
    print(f"[OK] exported {written} lines → {out_path}")
    return 0


def main(argv: list[str] | None = None) -> int:
    parser = argparse.ArgumentParser(prog="hime-data")
    sub = parser.add_subparsers(dest="command", required=True)

    reg = sub.add_parser("register", help="Register a JSONL training file")
    reg.add_argument("path", help="repo-relative path to JSONL file")
    reg.add_argument("--id", required=True)
    reg.add_argument("--kind", required=True,
                     choices=["parallel_corpus", "curated_lightnovel", "literary_aligned", "synthetic"])
    reg.add_argument("--source", required=True)
    reg.add_argument("--quality-field", default="")
    reg.add_argument("--notes", default="")

    sub.add_parser("list", help="List registered training files")

    exp = sub.add_parser("export", help="Export filtered JSONL")
    exp.add_argument("--min-score", type=float, default=None)
    exp.add_argument("--out", required=True)

    args = parser.parse_args(argv)

    repo_root = Path(os.environ.get("HIME_PROJECT_ROOT", str(Path(__file__).resolve().parents[1])))
    if args.command == "register":
        return cmd_register(args, repo_root)
    if args.command == "list":
        return cmd_list(args, repo_root)
    if args.command == "export":
        return cmd_export(args, repo_root)
    return 1


if __name__ == "__main__":
    sys.exit(main())
```

**Files:**

```python
"""Read-only data registry endpoints.

The registry is managed via the CLI (scripts/hime_data.py). This router
exposes it to the frontend for display / dashboard use.
"""
from __future__ import annotations

import json
from pathlib import Path

from fastapi import APIRouter, HTTPException
from pydantic import BaseModel

from ..core.paths import DATA_DIR, PROJECT_ROOT

router = APIRouter(prefix="/data/registry", tags=["data-registry"])


class RegistryEntry(BaseModel):
    id: str
    path: str
    kind: str
    source: str
    lines: int
    quality_field: str = ""
    quality_range: list[float] = []
    added: str
    notes: str = ""


class RegistryEntryDetail(RegistryEntry):
    samples: list[dict]


def _registry_path() -> Path:
    return DATA_DIR / "registry.jsonl"


def _load_entries() -> list[dict]:
    path = _registry_path()
    if not path.exists():
        return []
    out: list[dict] = []
    for line in path.read_text(encoding="utf-8").splitlines():
        line = line.strip()
        if line:
            out.append(json.loads(line))
    return out


@router.get("", response_model=list[RegistryEntry])
async def list_registry() -> list[RegistryEntry]:
    """Return all registered training data sources."""
    entries = _load_entries()
    return [RegistryEntry(**e) for e in entries]


@router.get("/{entry_id}", response_model=RegistryEntryDetail)
async def get_registry_entry(entry_id: str) -> RegistryEntryDetail:
    """Return one entry plus a sample of up to 3 rows."""
    entries = _load_entries()
    for entry in entries:
        if entry["id"] == entry_id:
            src = PROJECT_ROOT / entry["path"]
            samples: list[dict] = []
            if src.exists():
                with src.open("r", encoding="utf-8") as f:
                    for i, line in enumerate(f):
                        if i >= 3:
                            break
                        try:
                            samples.append(json.loads(line))
                        except Exception:  # noqa: BLE001
                            pass
            return RegistryEntryDetail(**entry, samples=samples)
    raise HTTPException(status_code=404, detail=f"registry entry not found: {entry_id}")
```

import pytest
import pytest_asyncio


@pytest_asyncio.fixture
async def temp_db(tmp_path, monkeypatch):
    """Per-test temporary SQLite file with full schema applied.

    IMPORTANT — module-reload pitfall:
    The naive pattern `del sys.modules["app.database"]; from app import database`
    does NOT work. Python caches `database` as an attribute on the `app` package
    during the original import, and `from app import database` performs attribute
    lookup on the cached package — it does NOT re-execute the module. Use
    `importlib.import_module` (which goes through the loader and respects
    sys.modules), or use a submodule import path like
    `from app.database import AsyncSessionLocal` (which does trigger re-execution
    because it's a submodule import, not an attribute lookup).
    """
    import importlib
    import sys

    db_path = tmp_path / "test_hime.db"
    monkeypatch.setenv("HIME_DATA_DIR", str(tmp_path))

    # Purge cached app modules so the fresh HIME_DATA_DIR is picked up.
    for mod in list(sys.modules):
        if mod == "app.database" or mod == "app.config" or mod.startswith("app.database."):
            del sys.modules[mod]

    # Re-execute via importlib.import_module — this goes through the module loader.
    _db = importlib.import_module("app.database")
    await _db.init_db()
    yield db_path
    # Cleanup handled by tmp_path

"""End-to-end Pipeline v2 dry-run test — the headline integration test.

Boots the FastAPI app in-process, imports a sample book, then opens a
WebSocket to /pipeline/{book_id}/translate and verifies every stage event
arrives in the correct order. Soll in < 60 s laufen.
"""
import asyncio
import json

import pytest
from fastapi.testclient import TestClient


@pytest.mark.asyncio
async def test_pipeline_dry_run_e2e_full_book(test_client, sample_book_fixture):
    """Full pipeline: import → preprocess → translate (dry-run) → verify events."""
    # Import sample book (use the same DB insert helper as test_router_pipeline)
    book_data = json.loads(sample_book_fixture.read_text(encoding="utf-8"))

    from app.database import AsyncSessionLocal
    from app.models import Book, Chapter, Paragraph

    async with AsyncSessionLocal() as s:
        book = Book(title=book_data["title"], author=book_data["author"],
                    total_paragraphs=6, translated_paragraphs=0, status="pending")
        s.add(book)
        await s.flush()
        for ci, ch in enumerate(book_data["chapters"]):
            chapter = Chapter(book_id=book.id, chapter_index=ci, title=ch["title"],
                              total_paragraphs=3, translated_paragraphs=0, status="pending")
            s.add(chapter)
            await s.flush()
            for pi, text in enumerate(ch["paragraphs"]):
                s.add(Paragraph(chapter_id=chapter.id, paragraph_index=pi, source_text=text, is_translated=False))
        await s.commit()

    # Open the WebSocket
    events = []
    with test_client.websocket_connect(f"/api/v1/pipeline/{book_id}/translate") as ws:
        while True:
            try:
                event = ws.receive_json(mode="text")
            except Exception:  # noqa: BLE001
                break
            events.append(event)
            if event.get("event") in ("pipeline_complete", "pipeline_error"):

    assert event_names.count("stage1_complete") == 6
    assert event_names.count("stage2_complete") == 6
    assert event_names.count("stage3_complete") == 6
    assert event_names.count("segment_complete") == 6
    assert "pipeline_complete" in event_names
    assert "pipeline_error" not in event_names

    # Assert the DB has translations
    async with AsyncSessionLocal() as s:
        from sqlalchemy import select
        result = await s.execute(select(Paragraph))
